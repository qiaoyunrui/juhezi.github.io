Groovy 中的柯里化闭包

带有预绑定形参的闭包叫做柯里化闭包（Curried Closure）。当对一个闭包调用curry()时，就是要求预先绑定某些形参。在预先绑定一个形参之后，调用闭包时就不必重复为这个形参提供实参了。

可以使用curry()方法柯里化任意多个参数，但这些形参必须是从前面开始的连续若干个。也就是说，如果有 n 个形参，可以任意柯里化前 k 个，其中 0 <= k <= n。

如果想柯里化后面的形参，可以使用 rcurry() 方法。如果想柯里化位于形参列表中间的形参，可以使用 ncurry() 方法，传入要进行柯里化的形参的位置，同时提供相应的值。
