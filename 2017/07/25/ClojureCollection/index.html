Clojure 中的集合与数据结构

map、vector、set 和列表是 Clojure 提供的基本数据类型。Clojure 为它们提供了便利的字面量：

  • ‘(a b :name 12.5) ;; 列表
  • [‘a ‘b :name 12.5] ;; vector
  • {:name “Hello” :age 31} ;; map
  • #{1 2 3} ;; set

Clojure 中的数据结构有两个特色：

  1. 数据结构首先是依据抽象来用的，而不是依据具体的的实现细节来用的。

  2. 数据结构是不可改变的而且是持久的
