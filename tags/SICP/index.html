丘奇(Church)计数

丘奇计数其实就是一种计数方法，建立在 lambda 演算的基础之上。 Code 补一个加法操作12345;((a f) x) 执行 A 的执行过程;要返回一个 fn，所以是固定格式(defn add [a b] (fn [f] (fn [x] ((b f) ((a f) x))))) Church 计的数，其实指的是 f 应用于 x 的次数。
